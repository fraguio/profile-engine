# CV Automation

Proyecto personal para automatizar la generación de mi curriculum vitae.

## Arquitectura:

- **JSON Resume (`resume.json`)** como fuente de verdad  
- Script Python para convertir a **RenderCV YAML**  
- **RenderCV** para generar salidas (PDF, Markdown, HTML, PNG)

## Condiciones del proyecto:

- Desarrollo obligatorio en **Dev Container**  
- Validación estricta mediante los schemas oficiales de entrada y salida  
- Enfoque incremental: cambios pequeños, trazables y alineados con un roadmap claro
